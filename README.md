# SCOUT-PARSER-BOT
A multifunctional Telegram bot built with Aiogram 3.x.
# Features:
Server monitoring (CPU/RAM/Uptime).
Crypto price tracking (Bitcoin).
IP-OSINT lookup.
Web price scraping.
# Setup:
Clone this repo: git clone ...
Create .env file from .env.example.
Install dependencies: pip install -r requirements.txt
Run: python3 main.py
